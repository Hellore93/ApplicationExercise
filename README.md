
## Quickstart

```
  npm install
  npm run init-project
  npm run watch
```

## Available Commands

1. `npm run watch` - starts the development server with hot reloading enabled

2. `npm run test` - start the test runner

3. `npm run lint` - runs linter to check for lint errors
